MiniLight

Without the fuss of every other syntax highlighter. Just like this website (Or rather webpage), installation and usage too is simple.

Let's have a look : 

// This is a Code Block themed on Sublime Text's Mariana Color Scheme.

/*
	How about another comment.
	This time a multiline one.
*/

let x = 5;

function increment(){
	return x + 1;
}

console.log(x + " - " + typeof x);	// The String is also highlighted.

Note : The script will fail at certain points, just be patient and file an Issue at the Github Repo.
